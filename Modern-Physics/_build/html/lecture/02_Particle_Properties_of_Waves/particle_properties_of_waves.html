2.4 x射線 X-Rays#

在1895年時，崙琴 (Wilhelm Roentgen) 發現當 快速電子撞擊物質時，會產生一種未知性質的 高穿透力輻射。 x 射線的產生 \lambda = \frac{1.2410^-6 mev}{E}

[x 射線的產生圖示:x射線管] x 射線管。選定一個金屬作為靶材，加速電壓 V 越高時，電子速度越快且 x 射線波長越短

[金屬鎢和在不同電位下的x射線光譜圖示] [金屬鎢和金屬鉬在35kV電位下的x射線光譜圖示]

發現金屬鉬的 x 射線光譜中有兩條明顯的特徵峰，分別稱為 Kα 和 Kβ 線，這些峰對應於電子從不同能階躍遷到 K 層所釋放的能量。這些特徵峰的存在證明了原子內部結構的存在，並且提供了對原子結構的深入理解。

發現x射線的穿透力和能量成正比，且和波長成反比。當電壓越高時，x射線的能量越大，波長越短，穿透力也越強。

[x 射線在醫學上的應用圖示:骨頭和肌肉的對比] 在現代的 x 射線管中，環繞油把熱從靶中帶走，並透過熱交換器釋放至外面的空氣。 使用 x 射線做為醫學上診斷的工具是利用不同的組織吸收不同的能量。因為骨頭中 含有鈣，對於 x 射線來說骨頭比肌肉較為不透明，而肌肉又比脂肪不透明。為加強 對比，給予病人含鋇食物可使其消化系統更加明顯，而其他化合物也可注入血液中 使得血管的狀況可供研究。 在電腦斷層掃描(Computerized Tomography) 中，從不同方向對病人照射一連串 的 x 射線，並利用電腦組成身體待檢查部分的橫截面影像。經由 x 射線照射，身 體組織等效於被電腦切成薄片來分析。任何想要的薄片都可以顯示出來。這個技 術使得異常組織可以被檢測出來且確定位置所在，這對於普通 x 射線圖來說可能 是做不到的（斷層這個字是源自於希臘字 tomos，即「切」的意思）。

Example 以 100 kV 電壓加速電子撞擊鎢靶，求出產生的 x 射線的最短波長。 電子的動能 \(E = eV = 100 \text{ kV} = 100 \times 10^3 \text{ V} \times 1.602 \times 10^{-19} \text{ C} = 1.602 \times 10^{-14} \text{ J}\) x 射線的最短波長 \(\lambda = \frac{1.24 \times 10^{-6} \text{ m} \cdot \text{eV}}{E \text{ (in eV)}} = \frac{1.24 \times 10^{-6} \text{ m} \cdot \text{eV}}{100 \text{ kV} \times 10^3 \text{ eV/kV}} = 1.24 \times 10^{-11} \text{ m}\)

此波長對應到的頻率 \(\nu = \frac{c}{\lambda} = \frac{3.00 \times 10^8 \text{ m/s}}{1.24 \times 10^{-11} \text{ m}} = 2.42 \times 10^{19} \text{ Hz}\)

2.5 x ray的繞射 (Diffraction of X-Rays)#

重要概念:如果要有interaction則兩個物體(電磁波波長、物體直徑)的特徵尺度必須相當

x 射線的波長和原子核的尺度相當，因此 x 射線可以被用來探測物質的原子結構。當 x 射線照射到晶體上時，會發生繞射現象，形成特定的繞射圖樣。這些圖樣可以被分析以揭示晶體內部的原子排列和結構。

[放一張x ray繞射圖示]

晶體中的原子可被用來定義許多平行平面族， ，每一個平面族之間具有特定的 間隔，這個分析是在 1913 年時由布拉格所提 出，而為了紀念他，我們將上述的平面稱為布 拉格平面(Bragg plane)。 2d sin θ = nλ, n = 1, 2, 3,… 其中 d 是布拉格平面之間的距離，θ 是入射 x 射線與布拉格平面之間的夾角，n 是繞射的階數，λ 是 x 射線的波長。

[NaCl晶體的兩種布拉格平面圖示] [立方晶體的 x 射線散射圖示] [NaCl 晶體的離子中散射的 x 射線所產生的干涉圖案]

[x 射線光譜儀圖示] x射線先通過準直儀(collimator)以確保入射的 x 射線是平行的，然後照射到晶體上，最後由探測器(detector)接收繞射後的 x 射線。通過測量繞射角度和強度，可以分析出晶體的結構和原子排列。 在 x 射線光譜儀中，x 射線照射到晶體上，並且在特定的角度下產生繞射。通過測量繞射角度和強度，可以分析出晶體的結構和原子排列。

2.6 康普頓效應 Compton Effect#

當 x 射線照射到物質上時，除了繞射之外，還會發生康普頓效應(Compton effect)，這是一種光子與電子之間的散射現象。當 x 射線光子與物質中的電子碰撞時，光子會將部分能量轉移給電子，導致光子的波長增加（能量減少）。這個效應提供了對光子粒子性的證據，並且是量子力學的重要實驗基礎之一。 ##重點:滿足牛頓力學的碰撞過程，光子和電子在碰撞過程中滿足動量守恆和能量守恆

[放一張康普頓效應圖示] 康普頓效應的數學描述可以通過康普頓散射公式來表達： $\(\lambda' - \lambda = \frac{h}{m_e c} (1 - \cos \phi)\)\( 其中 \)\lambda\( 是入射光子的波長，\)\lambda’\( 是散射後光子的波長，\)h\( 是普朗克常數，\)m_e\( 是電子的質量，\)c\( 是光速，\)\phi\( 是光子散射角度，\)\theta$ 是電子散射角度。

進一步驗證光子模型#

光子損失的能量 = 電子得到的能量 : $\(h\nu - h\nu' = \frac{1}{2} m_e v^2\)$

無質量粒子的動量和其能量相關，藉由下列公式 $\(p = \frac{E}{c}\)\( 可以得出光子動量的表達式 \)\(p = \frac{h\nu}{c}\)$

  • For X-axis: 初始動量 = 最終動量 (動量守恆) 與電子散射角度 \(\theta\) 的關係? 與光子的關係是?

\[\frac{h\nu}{c}+0 = \frac{h\nu'}{c}\cos \phi + m_e v \cos \theta\]

  • For Y-axis: 初始動量 = 最終動量 (動量守恆)

\[0 = \frac{h\nu'}{c}\sin \phi - m_e v \sin \theta\]
\[E=KE+mc^2=\sqrt{(pc)^2+(mc^2)^2}\]

先消除\theta，得到電子的速度 v 與光子散射角度 \phi 的關係，再將 v 代入動量守恆方程式中，最後得到康普頓散射公式

\[p^2c^2 = (h\nu - h\nu')^2 + 2m_e c^2 (h\nu - h\nu')\]

最後得到康普頓散射公式 $\(\lambda' - \lambda = \frac{h}{m_e c} (1 - \cos \phi)\)\( 定義康普頓波長 \)\lambda_c = \frac{h}{m_e c}\(，當 \)\phi = 180^\circ\( 時，\)\lambda’ - \lambda\( 的最大值為 \)\lambda_c$。

物理意義: 光子與電子的碰撞導致光子能量減少（波長增加），同時電子獲得動能，電子被散射的角度會影響光子能量的變化(是誰決定誰?)。這個效應證明了光子具有粒子性質，並且是量子力學的重要實驗基礎之一。

能量是否守恆? 答案是肯定的，因為光子損失的能量等於電子獲得的動能，這符合能量守恆定律。 是光子決定電子的散射角度，還是電子決定光子的散射角度? 答案是兩者互相影響，光子和電子在碰撞過程中會同時改變彼此的動量和能量，因此散射角度是由兩者的交互作用決定的。 但是實驗觀察到的\phi分布是均勻的，這意味著光子和電子的散射角度沒有特定的偏好，這也符合量子力學中粒子行為的隨機性。是否正確? 還是說\phi分布是由電子的初始狀態決定的? 還是說\phi分布是由光子的初始狀態決定的? 還是說\phi分布是由兩者的初始狀態共同決定的? 這個問題我也不確定。

[放一張康普頓散射的實驗圖示 相對強度比上波長在不同角度\phi的關係圖示] 康普頓散射的實驗確認，根據式 (2.21)，散射角度越大，則波長的變化越大。

Example 以 0.1 nm 波長的 x 射線照射到靶材上，當散射角度 \(\phi\) 為 90 度時，求出散射後光子的波長 \(\lambda'\)。求出在散射 x 射線中的極大波長，求出反彈電子的極大動能

康普頓波長 \(\lambda_c = \frac{h}{m_e c} = 2.43 \times 10^{-12} \text{ m}\)

波長變化 \(\lambda' - \lambda = \lambda_c (1 - \cos \phi) = 2.43 \times 10^{-12} \text{ m} \times (1 - \cos 90^\circ) = 2.43 \times 10^{-12} \text{ m}\)

散射後光子波長 \(\lambda' = \lambda + (\lambda' - \lambda) = 0.1 \text{ nm} + 2.43 \times 10^{-12} \text{ m} = 0.10243 \text{ nm}\)

反彈電子的極大動能

\[K_{\text{max}} = h\nu - h\nu' = \frac{hc}{\lambda} - \frac{hc}{\lambda'} = hc \left(\frac{1}{\lambda} - \frac{1}{\lambda'}\right)\]
\[K_{\text{max}} = (6.626 \times 10^{-34} \text{ J} \cdot \text{s})(3.00 \times 10^8 \text{ m/s}) \left(\frac{1}{0.1 \times 10^{-9} \text{ m}} - \frac{1}{0.10243 \times 10^{-9} \text{ m}}\right)\]
\[K_{\text{max}} \approx 1.94 \text{ keV}\]

如果x射線不去撞電子而是去撞原子核，會發生什麼事? 答案是會發生核散射(nuclear scattering)，這是一種光子與原子核之間的散射現象。當 x 射線光子與原子核碰撞時，光子會將部分能量轉移給原子核，導致光子的波長增加（能量減少）。這個效應提供了對光子粒子性的證據，並且是量子力學的重要實驗基礎之一。跟電子散射相比，核散射的效應通常較小，因為原子核的質量遠大於電子的質量，因此光子與原子核的碰撞會導致較小的能量轉移和波長變化。

x射線波長~10^-10 m 晶格大小~10^-10 m 核大小~10^-15 m 電子大小~10^-18 m 因此x射線的波長和晶格大小相當，可以發生繞射 x射線的波長遠大於核大小，無法發生繞射，但可以發生散射，這就是康普頓效應的來源 x射線的波長遠大於電子大小，無法發生繞射，但可以發生散射，這也是康普頓效應的來源

散射和繞射的區別，散射是指光子與物質中的粒子（如電子或原子核）發生碰撞，導致光子能量和方向的改變；繞射是指光子在遇到障礙物或通過狹縫時，波前發生彎曲和干涉現象，形成特定的圖樣。散射通常涉及能量轉移，而繞射則主要涉及波的干涉和衍射效應。

無線電波為什麼不會和牆體發生散射?如果是因為無線電特徵尺度長於牆體，那麼x射線的特徵長於核，為什麼還是會和核發生散射? 答案是無線電波的波長遠大於牆體的特徵尺度，因此無線電波無法與牆體發生有效的散射；而x射線的波長雖然遠大於核的特徵尺度，但仍然可以與核發生散射，這是因為x射線具有足夠的能量來與核中的粒子（如質子和中子）進行相互作用，導致能量轉移和波長變化

2.7 配對產生 Pair Production#

在碰撞中一個光子會將其全部（光電效應）或部分（康普頓效應）能量給予電子，對一個光子而言也可能轉化成一個電子和一個正子（帶正電的電子）也是有可能的，這個過程稱為配對產生(pair production)，電磁能量會轉換為物質。

當光子具有足夠的能量時，可以在核附近產生一對粒子和反粒子，這個過程稱為配對產生(pair production)。例如，一個高能光子可以在核附近產生一對電子和正電子。這個過程需要滿足能量守恆和動量守恆的條件，並且通常需要光子的能量至少為 1.022 MeV（兩倍的電子靜止能量）才能發生。

[放一張配對產生的圖示]

[電子－正子對形成的氣泡室圖] 電子－正子對形成的氣泡室圖，一個垂直於頁面的磁場使得電子和正子分別以 相反的曲線運動，此曲線路徑為螺旋狀，因為粒子在空間中運動時會損失能量。 在一個氣泡室中，液體（此處為氫）處於能使其保持為液體的壓力下被加熱超 過其沸點，隨後壓力釋放，氣泡會在不穩定過熱液體中的任意離子附近形成， 此時液體運動的帶電粒子將會留下氣泡痕跡，使得我們可以繪製成圖形。

Example 一個能量為 2 MeV 的光子在核附近發生配對產生，產生一對電子和正電子。求出這對粒子各自的動能。 光子能量 \(E_{\gamma} = 2 \text{ MeV}\) 電子和正電子的靜止能量 \(E_{e} = 0.511 \text{ MeV}\) 配對產生所需的最小能量 \(E_{\text{min}} = 2 \times E_{e} = 1.022 \text{ MeV}\)

Example 證明配對產生不會在空的空間中發生

光子能量 \(E_{\gamma}\) 必須至少為 1.022 MeV 才能產生一對電子和正電子。然而，在空的空間中，沒有核或其他粒子可以提供必要的動量轉移來滿足動量守恆。因此，即使光子具有足夠的能量，也無法在空的空間中發生配對產生，因為缺乏必要的條件來同時滿足能量守恆和動量守恆。

從能量守恆來看 \(h\nu = 2\gamma m_e c^2 + KE_e + KE_p\)

因為動量在橫切方向上必須守恆，故角度θ相等 \(p_e \sin \theta - p_p \sin \theta = 0\)

\(p_e \cos \theta + p_p \cos \theta = h\nu / c\)

因此可以得出\(p_e = p_p = h\nu / (2c)\)

因此可以得出\(KE_e = KE_p = \sqrt{(h\nu /2)^2 + (m_e c^2)^2} - m_e c^2\)

\(h\nu=2\gamma m_e c^2 (v/c)\cos \theta\) 因為v/c<1 ,cos theta<1,所以h\nu<2\gamma m_e c^2 但是配對產生需要h\nu>=2 m_e c^2 因此配對產生不會在空的空間中發生，需要有核或其他粒子來提供必要的動量轉移來滿足動量守恆。(核的能量被借走了，最後又還回來了?) (核的動量變多?)

這個過程如果沒有核會導致必然違背能量守恆或動量守恆，但是旁邊有核子就可以借走一些動量，讓配對產生的過程同時滿足能量守恆和動量守恆，因此配對產生不會在空的空間中發生，需要有核或其他粒子來提供必要的動量轉移來滿足動量守恆。(也可以借走一些能量嗎? 還是說核只是提供動量轉移，能量還是由光子提供的?)

Example 當一個電子和一個正子互相結合而滅絕時，在 +x 方向上並排以速度 0.500c 平行運動，沿著 x 軸產生兩個光子；(a)兩個光子都是沿著 +x 方向嗎？(b) 每個光子的能量為何？ (a) 兩個光子不會都是沿著 +x 方向，因為這樣會違反動量守恆。當電子和正子結合滅絕時，產生的兩個光子必須以相反的方向運動，以確保總動量為零。 (b) 每個光子的能量為 \(E_{\gamma} = m_e c^2 = 0.511 \text{ MeV}\)，因為電子和正子各自的靜止能量為 0.511 MeV，兩者結合後的總能量為 1.022 MeV，這些能量將轉化為兩個光子，每個光子攜帶 0.511 MeV 的能量。

或者 $\( p1-p2 = 2\gamma m_e v \)$

\[ p1c+p2c = 2\gamma m_e c^2 \]

解出 $\( p1+p2 = \)$

p1=0.885MeV/c p2=0.295MeV/c

E1=p1c=0.885MeV E2=p2c=0.295MeV

所以在正電子和電子相遇時會產生兩個光子?>那為什麼光子在沒有核子輔助時只需要在x軸上向前向後發射光子?

2.8 光子與重力 Photons and Gravity#

在一個重力場中降落的光子和石頭一樣會得到能量，這個能量的獲得會顯示在頻率由 v 增加至 v’ 的時候。

雖然光子沒有靜止質量，它們仍顯得像是具有重量一樣。 – 光子「質量」\(m=P/v=h \nu/c^2\)

光子在地球下降高度 H 之後的能量 hnu’=hnu(1+gH/c^2)

引力紅位移 (gravitational red shift)

\[\frac{\delta \nu}{\nu} =1-\frac{\nu'}{\nu} = \frac{GM}{c^2 R}\]

黑洞 (Black Hole) – 史瓦茲希德半徑 \(R_s = \frac{2GM}{c^2}\) – (factor 2 from general relativity correction)

Example 計算地球表面附近的引力紅位移，假設地球的質量為 \(M = 5.97 \times 10^{24} \text{ kg}\)，半徑為 \(R = 6.371 \times 10^6 \text{ m}\)，以及光速 \(c = 3.00 \times 10^8 \text{ m/s}\)。1960年時，下降光子的能量增加首度由龐德(Pound)和瑞布卡(Rebka)在哈佛大學觀察到。在他們的研究中， H 為 22.5m，求出頻率7.3*10^14 Hz的紅光子從 22.5 m 處降落之後，頻率變化為何。

\[ \nu'-\nu=(gH/c^2)\nu=(9.8 m/s^2 * 22.5 m / (3.00 \times 10^8 m/s)^2) \nu = 2.45 \times 10^{-15} \nu = 1.8Hz \]

4.1 核原子 (The Nuclear Atom)#

JJ湯姆生1898年說:核大小>>電子大小

拉賽福

  1. 推翻水果蛋糕模型

  2. 正電在小區域(原子核)

  3. data follow 方程式

  4. 核 10^-15 m 、 rho ~2.4*10^17 kg/m^3 ~ 中子星

拉塞福散射公式(Rutherford Scattering Formula)

\[N(\theta) = \frac{N_i nt Z^2 e^4}{(8\pi\epsilon_0)^2 r^2 KE^2 \sin^4(\theta/2)}\]

  • \(N(\theta)\) = number of \(\alpha\) particles per unit area reaching the screen at a scattering angle \(\theta\)

  • \(N_i\) = total number of \(\alpha\) particles reaching the screen

  • \(n\) = number of atoms per unit volume in the foil

  • \(Z\) = atomic number of the foil atoms

  • \(r\) = distance between the screen and the foil

  • \(KE\) = kinetic energy of the \(\alpha\) particles

  • \(t\) = foil thickness

pf#

\Delta \mathbf{p} = \mathbf{p}2 - \mathbf{p}1 = \int \mathbf{F} , dt \quad (4.24) \Delta p = 2mv \sin \frac{\theta}{2} \quad (4.25) \left| \int \mathbf{F} , dt \right| = \int F \cos \phi , dt \quad (4.26) 2mv \sin \frac{\theta}{2} = \int{-(\pi-\theta)/2}^{+(\pi-\theta)/2} F \cos \phi \frac{dt}{d\phi} , d\phi \quad (4.27) 2mv^2 b \sin \frac{\theta}{2} = \int{-(\pi-\theta)/2}^{+(\pi-\theta)/2} Fr^2 \cos \phi , d\phi \quad (4.28)

\cot \frac{\theta}{2} = \frac{4\pi\epsilon_0 KE}{Ze^2} b

\textbf{截面積} \quad \sigma = \pi b^2

f = \pi nt \left( \frac{Ze^2}{4\pi\epsilon_0 KE} \right)^2 \cot^2 \frac{\theta}{2}

df = -\pi nt \left( \frac{Ze^2}{4\pi\epsilon_0 KE} \right)^2 \cot \frac{\theta}{2} \csc^2 \frac{\theta}{2} d\theta

Head-on collision

\[KE_{\text{initial}} = PE = \left( \frac{1}{4\pi\epsilon_0} \right) \left( \frac{2Ze^2}{R} \right)\]

實驗上是 剛好燒壞靶材的前一刻的動能 跑到裡面剛好沒有動能

原子核的尺寸

\[R = \frac{2Ze^2}{4\pi\epsilon_0 KE_{\text{initial}}}\]

4.2 電子軌道 Electron Orbits#

Force Balance

\(F_c = F_e\)

\(\frac{mv^2}{r} = \frac{1}{4\pi\epsilon_0} \frac{e^2}{r^2}\)

Electron Speed

\(v_e = \frac{e}{\sqrt{4\pi\epsilon_0 mr}}\)

  • Kinetic Energy KE = 1/2 mv2

  • Potential Energy PE = - e^2 / 4πε0r

  • E = PE + KE = e^2 / 8πε0r - e^2 / 4πε0r

  • 氫原子的總能量

    • \(E = -\frac{e^2}{8\pi\epsilon_0 r}\)

和古典行星模型不一樣的地方是電子軌道模型有帶電會減速，會輻射出能量

例題 4.2#

實驗顯示需要13.6 eV的能量才能將氫原子的質子和電子分離；也就是說，它的總能量為 E = -13.6eV。求出氫原子的軌道半徑和電子速度。

\[r= -\frac{e^2}{8\pi \epsilon_0 r} \sim 5.3*10^-11 m\]
\[v = \frac{e}{\sqrt{4\pi \epsilon_0 mr}} \sim 2.2*10^6 m/s\]

計算v = 2*10^7m/s 的 alpha 粒子

\[\begin{split}\begin{aligned} \lambda = \frac{h}{mv} &= \frac{6.63 \times 10^{-34} \text{ J} \cdot \text{s}}{(6.6 \times 10^{-27} \text{ kg})(2.0 \times 10^7 \text{ m/s})} \\ &= 5.0 \times 10^{-15} \text{ m} \end{aligned}\end{split}\]

可以大概當作粒子(古典仍然適用)

百年前的實驗

萊曼系、巴莫、…帕申、布萊克、方德

\[\frac{1}{\lambda} = R \left( \frac{1}{2^2} - \frac{1}{n^2} \right) \quad n = 3, 4, 5, \dots\]
\[\frac{1}{\lambda} = R \left( \frac{1}{1^2} - \frac{1}{n^2} \right) \quad n = 3, 4, 5, \dots\]
\[\frac{1}{\lambda} = R \left( \frac{1}{3^2} - \frac{1}{n^2} \right) \quad n = 4, 5, \dots\]
\[\frac{1}{\lambda} = R \left( \frac{1}{4^2} - \frac{1}{n^2} \right) \quad n = 5,6 \dots\]
\[\frac{1}{\lambda} = R \left( \frac{1}{5^2} - \frac{1}{n^2} \right) \quad n = 6,7 \dots\]

光譜是元素的指紋，現在仍然在使用，看材料的種類 光譜不連續代表原子內能量不連續 放射光譜不一定嚴格等於吸收光譜，通常吸收光譜都比放射多

4.4 波耳原子 (The Bohr Atom)#

質子(r~10^-15m)應有波動性，假設行星模型正確

電子速度應該是: \(v = \frac{e}{\sqrt{4\pi\epsilon_0 mr}}\)

電子在H+的原子核環繞的德布羅伊波應該是: \(\lambda = \frac{h}{e} \sqrt{\frac{4\pi\epsilon_0 r}{m}}\)

要使能量不消失(Stable)擇要形成 駐波

\(n \lambda =2\pi r_n n=1,2,3,...\)

  • n是量子數

  • \lambda 是波長

  • r_n 是軌道半徑

可以得到

\(r_n = \frac{n^2 h^2 \epsilon_0}{\pi m e^2} \quad n = 1, 2, 3, \dots\)

波爾原子: 在r_n上電子才有stable structure

  • \(r_n\)：第 \(n\) 個能階的軌道半徑。

  • \(n\)：主量子數（Principal quantum number）。

  • \(h\)：普朗克常數。

  • \(\epsilon_0\)：真空電容率。

  • \(m\)：電子質量。

  • \(e\)：基本電荷。

\(a_0 = r1 = 5.292*10^-11m\)

\(rn=n^2 a_0\)

能階 (Energy Levels )#

仍假設行星正確 (那不正確會怎麼樣??)

\[E = -\frac{e^2}{8\pi\epsilon_0 r} \]
\[r_n = \frac{n^2 h^2 \epsilon_0}{\pi m e^2} \quad n = 1, 2, 3, \dots\]
\[E_n = -\frac{me^4}{8\epsilon_0^2 h^2} \left( \frac{1}{n^2} \right) = \frac{E_1}{n^2} \quad n = 1, 2, 3, \dots\]
\[E_1 = -2.18 \times 10^{-18} \text{ J} = -13.6 \text{ eV} \]

例題4.4.1

一個電子和一個基態的氫原子相撞並且將它激發到 的受激態。在這個非彈性碰撞（動能不守恆）中，氫原子獲得多少能量？

\[\Delta E = E_f - E_i = \frac{E_1}{n_f^2} - \frac{E_1}{n_i^2} = E_1 \left( \frac{1}{n_f^2} - \frac{1}{n_i^2} \right)\]
\[\Delta E = -13.6 \left( \frac{1}{3^2} - \frac{1}{1^2} \right) \text{ eV} = 12.1 \text{ eV}\]

例題4.4.2 在實驗室裡已經可以製造出處於高量子數狀態的氫原子並且可在太空中觀察到，這些氫原子稱為雷德堡原子(Rydberg atom)。(a) 求出一個半徑為0.0100 mm的波耳軌道氫原子量子數。(b) 在這個狀態下的氫原子能量為何？

(a) \(r_n = 1.00 \times 10^{-5} \text{ m} , \quad n = \sqrt{\frac{r_n}{a_0}} = \sqrt{\frac{1.00 \times 10^{-5} \text{ m}}{5.29 \times 10^{-11} \text{ m}}} = 435\)

(b) \(E_n = \frac{E_1}{n^2} = \frac{-13.6 \text{ eV}}{(435)^2} = -7.19 \times 10^{-5} \text{ eV}\)

  • 物理解釋: 雷德堡原子明顯地非常脆弱且容易被游離，這也是為什麼在自然的情況下，只有接近真空的空間才可以發現它們的存在。雷德堡原子光譜分布範圍至無線電波頻率，它們的存在便是透過無線電望遠鏡所觀測到的資料而確定的。

求出氫原子巴默系光譜中最長的波長，且對應到\(H_a\)的譜線。

\(\frac{1}{\lambda} = R \left( \frac{1}{n_f^2} - \frac{1}{n_i^2} \right) = R \left( \frac{1}{2^2} - \frac{1}{3^2} \right) = 0.139R\)

\(\lambda = \frac{1}{0.139R} = \frac{1}{0.139(1.097 \times 10^7 \text{ m}^{-1})} = 6.56 \times 10^{-7} \text{ m} = 656 \text{ nm}\)

4.5 對應原理 Correspondence Principle#

量子數越大時，量子物理越接近古典物理

古典#

\(Electron Speed: \)v = \frac{e}{\sqrt{4\pi\epsilon_0 mr}}\( \)Frequency: \(f = v / 2\pi r\) \(r_n = \frac{n^2 h^2 \epsilon_0}{\pi m e^2} \quad n = 1, 2, 3, \dots\) Frequency of Revolution: \(f = \frac{me^4}{8\epsilon_0^2 h^3} \left( \frac{2}{n^3} \right) = \frac{-E_1}{h} \left( \frac{2}{n^3} \right)\)

量子#

\[\begin{split} 1/ \lambda = E1/ch (1/nf^2-1/ni^2)\\ =E1/ch(1/(n-p)^2-1/n^2)\\ =-E1/h ((2np-p^2)/(n^2(n-p)^2))\\ (n \ll p) = -e1/n 2np/n^4 = -E1/n 2p/n^3\end{split}\]

p=1 時 量子=古典

例題4.5 (a) 求出 n = 1 和 n = 2 的波耳軌道旋轉頻率。(b) 當電子由n = 2 的軌道落到 n = 1 的軌道時，放出的光子頻率為何？(c)電子落到較低能階並放出光子之前停留在受激態的時間大約為 10-8 秒。在這段時間內，電子在 n = 2 的波耳軌道上轉了幾圈？

(a) f_1 = \frac{-E_1}{h} \left( \frac{2}{1^3} \right) = \left( \frac{2.18 \times 10^{-18} \text{ J}}{6.63 \times 10^{-34} \text{ J} \cdot \text{s}} \right)(2) = 6.58 \times 10^{15} \text{ rev/s}

f_2 = \frac{-E_1}{h} \left( \frac{2}{2^3} \right) = \frac{f_1}{8} = 0.823 \times 10^{15} \text{ rev/s}

rev是什麼單位?一圈? 這樣寫好嗎?或者是標準嗎

(b) \nu = -\frac{E_1}{h} \left( \frac{1}{n_f^2} - \frac{1}{n_i^2} \right) = \left( \frac{2.18 \times 10^{-18} \text{ J}}{6.63 \times 10^{-34} \text{ J} \cdot \text{s}} \right) \left( \frac{1}{1^2} - \frac{1}{2^2} \right) = 2.88 \times 10^{15} \text{ Hz}

(c) N = f_2 \Delta t = (8.23 \times 10^{14} \text{ rev/s})(1.00 \times 10^{-8} \text{ s}) = 8.23 \times 10^6 \text{ rev}

4.6 原子核運動 Nuclear Motion#

  • 原子核質量影響光譜線的波長，原子核越重這個效應越不明顯

  • 兩種粒子質量越接近放出的光能量越小

減縮質量 \(m' = \frac{mM}{m + M}\)

考慮原子核運動後修正的能階 \(E'_n = -\frac{m'e^4}{8\epsilon_0^2 h^2} \left( \frac{1}{n^2} \right) = \left( \frac{m'}{m} \right) \left( \frac{E_1}{n^2} \right) = \frac{m'}{m} E_n\)

這導致修正後的能階 \(E_n'\) 比原來的 \(E_n\) 稍微淺（絕對值變小）一點。 這個修正成功解釋了同位素位移（例如氫與氘的光譜差異）

例題4.6.1 氫原子 在氫原子中，電子質量為 \(m\)，原子核（質子）質量為 \(M\)。根據實驗測量：

  • \(m \approx 9.109 \times 10^{-31} \text{ kg}\)

  • \(M \approx 1.672 \times 10^{-27} \text{ kg}\)

  • 質子與電子的質量比 \(M/m \approx 1836.15\)

計算過程 減縮質量 \(m'\) 與電子質量 \(m\) 的比例公式為： $\(\frac{m'}{m} = \frac{\frac{mM}{m+M}}{m} = \frac{M}{m+M}\)\( 將分子分母同除以 \)m\(： \)\(\frac{m'}{m} = \frac{M/m}{1 + M/m}\)\( 代入數值： \)\(\frac{m'}{m} = \frac{1836.15}{1 + 1836.15} = \frac{1836.15}{1837.15} \approx \mathbf{0.99945}\)$

物理意義

  • 誤差極小：這個比例非常接近 \(1\)（誤差僅約 \(0.05\%\)），這說明波耳模型假設原子核靜止不動是一個非常好的近似。

  • 能階修正：修正後的能階 \(E'_n\) 會是原本的 \(0.99945\) 倍。雖然差異很小，但在高解析度的光譜實驗中是可以觀察到的。

  • 同位素對比：如果是氘原子（原子核含一個質子一個中子，\(M \approx 3670m\)），其 \(m'/m\) 約為 \(0.99973\)。氫與氘之間這微小的比例差異，就是發現氘（重氫）的關鍵證據。

例題4.6.2
正電子「原子」 為一個正電子(positron)和一個電子相互繞行的系統。試比較正電子原子的光譜線波長與一般氫原子光譜線的波長。

m’ = \frac{mM}{m + M} = \frac{m^2}{2m} = \frac{m}{2}

E’_n = \left( \frac{m’}{m} \right) \frac{E_1}{n^2} = \frac{E_1}{2n^2}

這表示雷德堡常數在正電子原子中為一般氫原子的一半。因此正電子原子光譜線所有的波長均為氫原子光譜線波長的兩倍

例題4.6.3

渺是一種不穩定的基本粒子，質量為電子質量(me)的207倍，帶一單位的正電或負電，一個負渺(mu )可以被一個原子核攫取而形成一個渺原子。(a) 一個質子攫取一個負渺。求出這個原子的第一波耳軌道半徑。(b) 求出這個原子的游離能

(a) \(m' = \frac{mM}{m + M} = \frac{(207m_e)(1836m_e)}{207m_e + 1836m_e} = 186m_e\)

\(r'_1 = \left( \frac{m}{m'} \right) r_1 = \left( \frac{m_e}{186m_e} \right) a_0 = 2.85 \times 10^{-13} \text{ m}\)

(b) E’_1 = \left( \frac{m’}{m} \right) E_1 = 186E_1 = -2.53 \times 10^3 \text{ eV} = -2.53 \text{ keV}

  • 渺比電子靠近質子，大約為 186 倍，因此一個包含渺的氫原子會遠小於一般的氫原子

  • 能量與質量成正比 (E 整比 m)。因為緲子系統的減縮質量是電子的 186 倍，所以其能階能量也是原本的 186 倍。

  • 光子能量位於 X 射線 甚至 gamma 射線 區域，而非可見光或紫外光。

七、轉換法中消除混疊之網格點數條件證明#

證明目標：

\[N \ge 3K + 1\]

假設與已知 (Assumptions & Preliminaries)#

  • 【已知 1】 物理空間中截斷最大波數為 \(K\) 的頻譜展開式：

    \[\begin{split}\begin{gather*} u(x) &=& \sum_{k_1=-K}^{K} \hat{u}_{k_1} e^{ik_1 x} \\ v(x) &=& \sum_{k_2=-K}^{K} \hat{v}_{k_2} e^{ik_2 x} \end{gather*}\end{split}\]

    • \(u(x)\) : 物理空間上的變數分量／東西向速度微擾場 (Physical zonal velocity component) \([\text{m}\cdot\text{s}^{-1}]\)

    • \(v(x)\) : 物理空間上的變數分量／南北向速度微擾場 (Physical meridional velocity component) \([\text{m}\cdot\text{s}^{-1}]\)

    • \(K\) : 頻譜截斷最大波數 (Truncated maximum wavenumber) \([\text{m}^{-1}]\)，代表數值模式中能解析的最高波數限制

    • \(k_{1}\) : 頻譜空間中對應 \(u\) 場的整數波數索引 (Wavenumber spectrum index for u) \([\text{m}^{-1}]\)

    • \(k_{2}\) : 頻譜空間中對應 \(v\) 場的整數波數索引 (Wavenumber spectrum index for v) \([\text{m}^{-1}]\)

    • \(\hat{u}_{k_{1}}\) : 頻譜空間中對應波數 \(k_{1}\) 的 \(u\) 場傅立葉振幅係數 (Fourier amplitude coefficient of u) \([\text{m}\cdot\text{s}^{-1}]\)

    • \(\hat{v}_{k_{2}}\) : 頻譜空間中對應波數 \(k_{2}\) 的 \(v\) 場傅立葉振幅係數 (Fourier amplitude coefficient of v) \([\text{m}\cdot\text{s}^{-1}]\)

    • \(i\) : 虛數單位 (Imaginary unit, \(i = \sqrt{-1}\)) [無單位]

    • \(e^{ik_{1}x}\) : 對應波數 \(k_{1}\) 的空間諧振基底函數 (Spatial harmonic basis function) [無單位]

    • \(e^{ik_{2}x}\) : 對應波數 \(k_{2}\) 的空間諧振基底函數 (Spatial harmonic basis function) [無單位]

  • 【已知 2】 轉換法中直接於物理空間計算的二次非線性交互作用項：

    \[F(x) = u(x)v(x)\]

    • \(F(x)\) : 物理空間上的二次非線性交互作用乘積項 (Quadratic nonlinear interaction term in physical space) \([\text{m}^2\cdot\text{s}^{-2}]\)，常代表大氣動力學中的非線性平流強迫力

    • \(u(x)\) : 物理空間上的變數分量／東西向速度微擾場 (Physical zonal velocity component) \([\text{m}\cdot\text{s}^{-1}]\)

    • \(v(x)\) : 物理空間上的變數分量／南北向速度微擾場 (Physical meridional velocity component) \([\text{m}\cdot\text{s}^{-1}]\)

  • 【已知 3】 離散梯形積分的正交性與混疊條件（Orthogonality and aliasing condition of discrete trapezoidal integration）：

    \[\begin{split}\frac{1}{N} \sum_{j=0}^{N-1} e^{i m x_j} = \begin{cases} 1, & m = pN \quad (p \in \mathbb{Z}) \\ 0, & m \neq pN \end{cases}\end{split}\]

    • \(N\) : 物理空間的總網格點數 (Total number of grid points in physical space) [無單位]，代表空間離散化的解析度限制

    • \(x_{j}\) : 物理空間中第 \(j\) 個離散網格點的幾何位置 (Spatial position of grid point \(j\)) \([\text{rad}]\) 或 \([\text{m}]\)

    • \(m\) : 進入積分函數內部求和的淨總和波數 (Net combined wavenumber) \([\text{m}^{-1}]\)，在頻譜非線性乘積展開中通常由多個交互作用波數相加減構成（如 \(m = k_1 + k_2 - k\)）

    • \(p\) : 任意整數，當 \(p \neq 0\) 且滿足 \(m=pN\) 時即發生高頻混疊（Aliasing）至低頻

  • 【定義 1】 目標更新保留之頻譜波數有效範圍：

    \[|k| \le K\]

    • \(k\) : 目標計算與保留的頻譜波數索引 (Target spectral wavenumber index) \([\text{m}^{-1}]\)，代表目前正在計算更新的特定傅立葉分量

    • \(|k|\) : 目標頻譜波數的絕對值 (Absolute value of the target wavenumber) \([\text{m}^{-1}]\)

    • \(K\) : 頻譜截斷最大波數 (Truncated maximum wavenumber) \([\text{m}^{-1}]\)，代表數值模式中所定義並保留的最高物理有效波數上限

  • 【定義 2】 內部總和波數代換：

    \[m \overset{\text{def}}{=} k_1 + k_2 - k\]

    • \(m\) : 進入離散梯形積分內部求和的淨總和波數 (Net combined internal wavenumber) \([\text{m}^{-1}]\)，由交互作用波數組合而成，用以判斷是否滿足正交或混疊條件

    • \(k_{1}\) : 頻譜空間中第一個物理變數微擾場（如 \(u\) 場）的傅立葉整數波數索引 (Fourier wavenumber index for the first variable) \([\text{m}^{-1}]\)

    • \(k_{2}\) : 頻譜空間中第二個物理變數微擾場（如 \(v\) 場）的傅立葉整數波數索引 (Fourier wavenumber index for the second variable) \([\text{m}^{-1}]\)

    • \(k\) : 目標計算與保留的頻譜波數索引 (Target spectral wavenumber index) \([\text{m}^{-1}]\)

  • 【推導 1】 非線性物理強迫項 \(F(x)\) 展開式：

    \[\begin{split}\begin{gather*} F(x) &\overset{\text{已知 2}}{=}& u(x)v(x) \\ F(x) &\overset{\text{已知 1}}{=}& \left( \sum_{k_1=-K}^{K} \hat{u}_{k_1} e^{ik_1 x} \right) \left( \sum_{k_2=-K}^{K} \hat{v}_{k_2} e^{ik_2 x} \right) \\ F(x) &=& \sum_{k_1=-K}^{K} \sum_{k_2=-K}^{K} \hat{u}_{k_1} \hat{v}_{k_2} e^{i(k_1+k_2)x} \end{gather*}\end{split}\]

    • \(F(x)\) : 物理空間上的二次非線性交互作用乘積項 (Quadratic nonlinear interaction term in physical space) \([\text{m}^2\cdot\text{s}^{-2}]\)，常代表大氣動力學中的非線性平流強迫力

    • \(u(x)\) : 物理空間上的變數分量／東西向速度微擾場 (Physical zonal velocity component) \([\text{m}\cdot\text{s}^{-1}]\)

    • \(v(x)\) : 物理空間上的變數分量／南北向速度微擾場 (Physical meridional velocity component) \([\text{m}\cdot\text{s}^{-1}]\)

    • \(k_{1}\) : 頻譜空間中第一個物理變數微擾場（如 \(u\) 場）的傅立葉整數波數索引 (Fourier wavenumber index for the first variable) \([\text{m}^{-1}]\)

    • \(k_{2}\) : 頻譜空間中第二個物理變數微擾場（如 \(v\) 場）的傅立葉整數波數索引 (Fourier wavenumber index for the second variable) \([\text{m}^{-1}]\)

    • \(k\) : 目標計算與保留的頻譜波數索引 (Target spectral wavenumber index) \([\text{m}^{-1}]\)

  • 【推導 2】 離散積分函數內最高波動頻率 \(\max(|m|)\) 之極值：

    \[\begin{split}\begin{gather*} \max(|m|) &\overset{\text{定義 2}}{=}& \max(|k_1 + k_2 - k|) \\ \max(|m|) &\le& \max(|k_1|) + \max(|k_2|) + \max(|-k|) \\ \max(|m|) &\overset{\text{已知 1, 定義 1}}{\le}& K + K + K \\ \max(|m|) &=& 3K \end{gather*}\end{split}\]

    • \(\max(|m|)\) : 進入離散積分時可能產生的最高淨總和波數絕對值 (Maximum absolute net wavenumber) \([\text{m}^{-1}]\)

    • \(k_{1}\) : 頻譜空間中第一個物理變數微擾場（如 \(u\) 場）的傅立葉整數波數索引 (Fourier wavenumber index for the first variable) \([\text{m}^{-1}]\)

    • \(k_{2}\) : 頻譜空間中第二個物理變數微擾場（如 \(v\) 場）的傅立葉整數波數索引 (Fourier wavenumber index for the second variable) \([\text{m}^{-1}]\)

    • \(k\) : 目標計算與保留的頻譜波數索引 (Target spectral wavenumber index) \([\text{m}^{-1}]\)

證明#

求取目標波數 \(k\) 的離散傅立葉轉換係數 \(\hat{F}_{k,\text{discrete}}\)：

\[\begin{split}\begin{gather*} \hat{F}_{k,\text{discrete}} &=& \frac{1}{N} \sum_{j=0}^{N-1} F(x_j) e^{-ik x_j} \\ \hat{F}_{k,\text{discrete}} &\overset{\text{推導 1}}{=}& \frac{1}{N} \sum_{j=0}^{N-1} \left( \sum_{k_1=-K}^{K} \sum_{k_2=-K}^{K} \hat{u}_{k_1} \hat{v}_{k_2} e^{i(k_1+k_2)x_j} \right) e^{-ik x_j} \\ \hat{F}_{k,\text{discrete}} &=& \frac{1}{N} \sum_{j=0}^{N-1} \sum_{k_1=-K}^{K} \sum_{k_2=-K}^{K} \hat{u}_{k_1} \hat{v}_{k_2} e^{i(k_1+k_2-k)x_j} \\ \hat{F}_{k,\text{discrete}} &\overset{\text{定義 2}}{=}& \sum_{k_1=-K}^{K} \sum_{k_2=-K}^{K} \hat{u}_{k_1} \hat{v}_{k_2} \left( \frac{1}{N} \sum_{j=0}^{N-1} e^{i m x_j} \right) \end{gather*}\end{split}\]

為確保精確萃取係數且無混疊誤差，必須保證上式中括號內的離散積分項，絕對不會觸發 \(p \neq 0\) 的非零投影條件：

\[\begin{split}\begin{gather*} \max(|m|) &<& |p|N \quad (p \in \mathbb{Z}, p \neq 0) \\ \max(|m|) &<& |1|N \\ \max(|m|) &<& N \\ 3K &\overset{\text{推導 2}}{<}& N \\ N &\ge& 3K + 1 \end{gather*}\end{split}\]

Matsuno (1966) vs. Gill (1980)#

長波近似的核心差異#

0. 色散 (Dispersion) 快速複習#

  • 色散波： \(\omega\) 與 \(k\) 非線性正比 → 相速度 \(c = \omega/k\) 隨波長改變 → 波包「散開」

  • 非色散波： \(c\) 與 \(k\) 無關（如淺水重力波 \(c = \sqrt{gH}\)）→ 波形不變形

  • 關鍵量：群速度 \(c_g = \partial\omega/\partial k\)（能量傳遞速度），色散波中 \(c \neq c_g\)

類比： 池塘丟石頭 → 長波快、短波慢 → 波紋「散開」= 色散。真空中的光 = 非色散。

2. 數學近似的差異#

完整色散關係（出發點相同）#

\[\omega^2 - k^2 - \frac{k}{\omega} - 1 = 2n\]

Matsuno：只砍 \(\omega^2\)（低頻近似）#

  • 假設：\(\omega \ll 1\)（低頻運動）→ 忽略 \(\omega^2\)

  • 結果：

\[\omega = \frac{-k}{k^2 + 2n + 1}\]

  • \(k^2\) 保留 → 羅斯貝波仍為色散波

    • 長波（小 \(k\)）：相速度快

    • 短波（大 \(k\)）：相速度極慢，近乎停滯

Gill：同時砍 \(\omega^2\) 與 \(k^2\)（嚴格長波近似）#

  • 假設：\(L_x \gg L_y\)（東西尺度遠大於南北）→ \(k \to 0\)

  • 動力學操作：南北向動量方程式中 \(\partial v/\partial t\) 直接砍掉 → 強迫地轉平衡

\[yu + \frac{\partial \phi}{\partial y} = 0\]

  • 結果：

\[\omega = \frac{-k}{2n + 1}\]

  • \(k^2\) 消失 → \(\omega \propto k\) → 羅斯貝波變為非色散波

    • 給定模態 \(n\)，所有波長一律以 \(c = -1/(2n+1)\) 向西傳播

3. 核心概念：\(k^2\) 項的命運#

“The fate of the \(k^2\) term” 是理解兩者差異的鑰匙。

  • Matsuno 保留 \(k^2\) → 物理完整，但數學複雜，難以求解析解

  • Gill 消滅 \(k^2\) → 物理略失真，但換來數學優雅：

    • 可用特徵線法 (Method of Characteristics) 求解

    • 畫出著名的 Gill Pattern：

      • 加熱源東側：赤道克耳文波下沉區

      • 加熱源西側：一對羅斯貝波氣旋式環流（雙胞胎氣旋）
